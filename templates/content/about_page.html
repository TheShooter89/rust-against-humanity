About this project

rust-against-humanity is a small self-educational experiment aimed to play around with Rust and htmx technologies


this web app is basically an online multiplayer clone of Cards Against Humanity card game

Backend

Server has been developed in Rust leveraging axum async runtime and askama templating engine

Frontend

Frontend is almost entirely prerendered (synchronously) on the server. AJAX calls are handled using htmx framework, combined with very few lines of Typescript bundled in a single file
